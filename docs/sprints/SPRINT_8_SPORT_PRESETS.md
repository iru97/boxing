# Sprint 8: Sport-Specific Preset & Session System

## Sprint Goal

Restructure the preset/session system from a flat list of generic number combinations into a sport-aware, category-organized hierarchy that reflects how athletes in Boxing, Muay Thai, MMA, BJJ, Kickboxing, and Wrestling actually train and compete.

---

## Research Findings

### 1. Boxing

**Competition formats:**
- Professional Men: 12 rounds x 3:00, 1:00 rest
- Professional Women: 10 rounds x 2:00, 1:00 rest
- Amateur Men: 3 rounds x 3:00, 1:00 rest
- Amateur Women: 3 rounds x 2:00, 1:00 rest
- Youth: 3 rounds x 1:30, 1:00 rest

**Training structure:**
A typical boxing session runs 60-90 minutes built around 3:00 on / 1:00 off. Session flow:
1. Warm-up: jump rope (3-5 rounds)
2. Shadow boxing: 3-5 rounds x 3:00 / 0:30 rest (lighter rest, technique focus)
3. Heavy bag: 6-10 rounds x 3:00 / 1:00 rest (power, volume, combos)
4. Speed bag: 3-6 rounds x 2:00 / 0:30 rest (rhythm, timing)
5. Pad work: 3-5 rounds x 3:00 / 1:00 rest (coach-led combinations)
6. Sparring: 4-8 rounds x 3:00 / 1:00 rest (fight simulation)
7. Conditioning: bodyweight circuits, core work

**What makes boxing training distinct:**
- Everything revolves around the 3-minute round
- Shadow boxing uses shorter rest (30s) because intensity is lower
- Speed bag uses shorter rounds (2:00) because it is endurance/rhythm focused
- Conditioning uses short bursts (20-30s) or EMOM format

### 2. Muay Thai

**Competition formats:**
- Professional (WBC/WMC): 5 rounds x 3:00, 2:00 rest (longer rest than boxing)
- Amateur 18+: 3 rounds x 2:00, 1:00 rest
- Amateur 14-17: 3 rounds x 1:30, 0:45 rest
- Amateur under 13: 3 rounds x 1:00, 0:30 rest

**Training structure:**
A typical Muay Thai session runs 90-120 minutes:
1. Run: 3-5 miles (done before session)
2. Jump rope: 3-5 rounds x 3:00
3. Shadow boxing: 3-5 rounds x 3:00 / 0:30 rest (includes kick shadow work)
4. Pad work: 5-8 rounds x 3:00 / 2:00 rest (Thai pads, kicks + knees + elbows + punches)
5. Heavy bag: 5-6 rounds x 3:00 / 1:00 rest (often includes 100 kick drills per round)
6. Clinch work: 3-5 rounds x 3:00 / 1:00 rest (stand-up grappling, knees)
7. Sparring: 3-5 rounds x 3:00 / 2:00 rest
8. Conditioning: 20-30 min (body weight, core, neck exercises)

**What makes Muay Thai training distinct:**
- Longer rest periods (2:00) mirror the competition format
- Clinch work is a unique training phase (stand-up grappling for knees and sweeps)
- Pad work is king -- Thai pad rounds are the core of training
- Kick-focused rounds: repetitive striking drills (100 kicks, 100 knees per round)
- Combination of 8 weapons (fists, elbows, knees, shins) requires specific drill structure

### 3. MMA

**Competition formats:**
- UFC/Professional non-title: 3 rounds x 5:00, 1:00 rest
- UFC/Professional title: 5 rounds x 5:00, 1:00 rest
- Amateur: 3 rounds x 3:00, 1:00 rest

**Training structure:**
MMA training is split across disciplines, not done all at once. Typical weekly split:
- Striking days: 2-3x/week (Muay Thai or boxing format rounds)
- Grappling days: 2-3x/week (BJJ/wrestling format)
- MMA sparring: 1-2x/week (full MMA rules, 5:00 rounds)
- Conditioning: 2-3x/week (energy system work, strength)

A single MMA sparring session:
1. Warm-up / shadow work: 2-3 rounds x 3:00
2. Technique drilling: 20-30 min (takedown entries, ground-and-pound, wall work)
3. MMA sparring: 3-5 rounds x 5:00, 1:00 rest
4. Positional rounds: 3-5 rounds x 3:00 (start from specific position)
5. Conditioning: HIIT intervals (6-14s work / 15-36s rest, reflecting MMA work-rest ratio)

**What makes MMA training distinct:**
- 5-minute rounds are the standard (much longer than boxing/MT)
- Cross-discipline training: days dedicated to specific areas
- Positional drilling starts from specific scenarios (against cage, on ground)
- Transition drills: striking-to-takedown, sprawl-to-strikes
- Energy system work matches MMA's 1:2 to 1:4 work-rest ratio

### 4. BJJ (Brazilian Jiu-Jitsu)

**Competition formats:**
- IBJJF White belt: 5:00 (continuous, no rounds)
- IBJJF Blue belt: 6:00
- IBJJF Purple belt: 7:00
- IBJJF Brown belt: 8:00
- IBJJF Black belt: 10:00
- ADCC Worlds qualifying: 10:00 + 5:00 overtime
- ADCC Worlds finals: 20:00 + 10:00 overtime
- ADCC Trials qualifying: 6:00 + 3:00 overtime
- ADCC Trials finals: 8:00 + 4:00 overtime

**Training structure:**
A typical BJJ class runs 60-90 minutes:
1. Warm-up: 10-15 min (shrimps, rolls, breakfalls, guard retention drills)
2. Technique instruction: 15 min (instructor demo of 2-3 techniques)
3. Drilling: 20-30 min (partner repetition of techniques shown)
4. Positional sparring: 15-20 min (start from specific position: guard, mount, side control, back)
5. Rolling (free sparring): 20-30 min (5-6 rounds x 5:00 or 6:00 with brief transitions)

**What makes BJJ training distinct:**
- No rounds in competition -- continuous match with a total time limit
- Training uses timed rolls (5:00 or 6:00 rounds) to simulate match duration
- Positional sparring: short rounds (2-3 min) from a set starting position
- Drilling is untimed or uses long low-intensity blocks (10-15 min of repetition)
- "Shark tank" / "king of the hill": one person stays in while fresh partners rotate every 2-3 min
- The timer serves a different purpose than in striking sports: it bounds free sparring rather than structuring rounds-and-rest

### 5. Kickboxing

**Competition formats:**
- K-1 Professional: 3 rounds x 3:00, 1:00 rest (title: 5 rounds)
- K-1 Amateur: 3 rounds x 2:00, 1:00 rest (World Championships: 2 rounds x 2:00, 0:30 rest)
- GLORY Professional: 3 rounds x 3:00, 1:00 rest (title: 5 rounds)
- WAKO/WKA Amateur: 3 rounds x 2:00, 1:00 rest

**Training structure:**
A typical kickboxing session runs 60-90 minutes:
1. Warm-up: 5-10 min (jump rope, dynamic stretches)
2. Shadow boxing/kicking: 3-5 rounds x 3:00 / 0:30 rest
3. Pad work: 4-6 rounds x 3:00 / 1:00 rest (focus mitts + shin pads)
4. Heavy bag: 4-6 rounds x 3:00 / 1:00 rest (flow rounds, power rounds, combo rounds)
5. Sparring: 3-5 rounds x 3:00 / 1:00 rest
6. Conditioning: HIIT, plyometrics

**What makes kickboxing training distinct:**
- Very similar to boxing but with kicks and sometimes knees (no elbows in K-1)
- Shin conditioning drills on bag (repetitive kicking rounds)
- Combination of boxing and kicking techniques in same rounds
- K-1 allows knees in clinch (one-step only) unlike full Muay Thai clinch

### 6. Wrestling

**Competition formats:**
- NCAA Collegiate (Folkstyle): 3 periods: 3:00 + 2:00 + 2:00 (total 7:00)
- High School (Folkstyle): 3 periods x 2:00 (total 6:00)
- Freestyle/Greco-Roman (International): 2 periods x 3:00, 0:30 break
- Youth (various): 2-3 periods x 1:00-2:00

**Training structure:**
A typical wrestling practice runs 90-120 minutes:
1. Warm-up/flexibility: 10-15 min (dynamic stretching, wrestling-specific movement)
2. Drill work: 10-15 min (passive and active drilling of techniques)
3. Technique instruction: 10-15 min (coach demonstrates, athletes practice step-by-step)
4. Situational wrestling: 15-20 min (start from specific position, work for set time)
5. Live wrestling: 30-60 min (full sparring, 2-3 min periods, or "shark bait" 30s periods in 3-4 person groups)
6. Conditioning: 10-15 min (sprints, carries, bodyweight circuits)

**What makes wrestling training distinct:**
- Period-based rather than round-based (folkstyle has asymmetric periods: 3+2+2)
- "Shark bait" format: 30-second bursts with rotation among 3-4 partners
- Drill work uses very short timed intervals (30s-60s per rep set)
- Situational wrestling starts from specific referee's position (top, bottom, neutral)
- Conditioning is brutally intense and often follows practice

### Updated `SessionModel`

Add two new fields to `SessionModel`:

```dart
@freezed
class SessionModel with _$SessionModel {
  const factory SessionModel({
    required String id,
    required String name,
    required int rounds,
    required int roundDurationSec,
    required int restDurationSec,
    @Default(10) int warningTimeSec,
    @Default(0) int warmupDurationSec,
    @Default(true) bool autoAdvance,
    @Default(true) bool keepScreenOn,
    @Default(false) bool voiceAnnounce,
    @Default(false) bool volumeOverride,
    @Default('classic_bell') String soundPack,
    @Default([]) List<RoundOverride> roundOverrides,
    @Default(false) bool isPreset,
    @Default(null) RoundTemplate? roundTemplate,
    @Default({}) Map<int, RoundTemplate> roundTemplateOverrides,
    // --- NEW FIELDS ---
    @Default(null) String? sport,       // 'boxing' | 'muayThai' | 'mma' | 'bjj' | 'kickboxing' | 'wrestling' | null
    @Default(null) String? category,    // Subcategory within sport: 'competition' | 'training' | 'drills' | null
  }) = _SessionModel;

  factory SessionModel.fromJson(Map<String, dynamic> json) =>
      _$SessionModelFromJson(json);
}
```

**Why `String?` instead of enums for the model fields**: The Freezed/JSON serialization works cleanly with strings. The `Sport` and `SessionCategory` enums are used in the UI/domain layer only. Custom user sessions can set `sport` and `category` to `null` or to a value if they want to associate with a sport.

## UI Changes

### Home Screen Restructure

The home screen currently uses `SessionCategory` enum with hardcoded ID-based mapping. It needs to be restructured for sport-first navigation.

#### Option: Sport Filter Chips + Collapsible Categories

The home screen will add a horizontal row of sport filter chips below the brand header. When a sport is selected, only that sport's presets are shown. "All" shows everything grouped by sport.

```
┌─────────────────────────────────────────┐
│  [History]                 [Settings]   │
│                                         │
│              B O X I N G                │
│           ──── ● ────                   │
│        TRAIN LIKE YOU FIGHT             │
│                                         │
│  [All] [Boxing] [MT] [MMA] [BJJ] ...   │  ← Sport filter chips (scrollable)
│                                         │
│  ┌ IN PROGRESS ─────────────────────┐   │
│  │ Heavy Bag  Round 3/8  2:47 left  │   │
│  │ [RESUME]              [DISCARD]  │   │
│  └──────────────────────────────────┘   │
│                                         │
│  MY SESSIONS                            │
│  ┌──────────────────────────────────┐   │
│  │ Custom Session 1                 │   │
│  └──────────────────────────────────┘   │
│                                         │
│  ▸ Competition (3)                      │  ← Collapsible category
│  ▸ Training (5)                         │
│  ▸ Drills (3)                           │
│                                         │
│  CONDITIONING                           │
│  ▸ Conditioning (3)                     │
│                                         │
└─────────────────────────────────────────┘
```

#### When a sport chip is selected:

- "All" chip: Groups by sport, each sport header is collapsible with sub-groups (Competition / Training / Drills)
- Specific sport chip (e.g. "Muay Thai"): Shows only that sport's presets grouped into Competition / Training / Drills sections
- Quick Start row shows most relevant presets for the selected sport (first 3 from Training category)

#### Sport Chip Design

Each chip shows the sport's accent color as a left-edge dot and the sport name:

```dart
FilterChip(
  label: Text('Muay Thai'),
  avatar: CircleAvatar(backgroundColor: SportColors.muayThai, radius: 5),
  selected: selectedSport == Sport.muayThai,
  onSelected: (val) => ref.read(selectedSportFilter.notifier).state =
      val ? Sport.muayThai : null,
)
```

#### Visual Identity Per Sport

Each sport's presets use the sport's color for:
- The vertical accent bar on session cards
- The category section header icon and label color
- The filter chip selected state

#### Category Labels Within a Sport

When viewing a specific sport, sections are labeled:
- "Competition" (with trophy icon)
- "Training" (with fitness icon)
- "Drills" (with timer/segments icon)

When viewing "All", sections are labeled by sport name, with Competition/Training/Drills as nested collapsible groups.

### Session Card Enhancement

The session card currently shows name, work/rest, total duration, and rounds. Add a subtle sport badge:

```
┌──────────────────────────────────────┐
│ ▌ Pro Muay Thai              5R ▶   │  ← Sport color bar
│ ▌ 3:00 / 2:00 rest · 25 min        │
│ ▌ MUAY THAI · COMPETITION           │  ← Sport + category label (subtle)
└──────────────────────────────────────┘
```

The sport + category label is shown in small text (bodySmall, 40% white opacity) below the duration info.

### Session Editor Enhancement

When creating a new custom session, add an optional "Sport" dropdown at the top:
- Options: Boxing, Muay Thai, MMA, BJJ, Kickboxing, Wrestling, (None)
- Selecting a sport pre-fills sensible defaults (e.g., selecting Muay Thai sets rest to 2:00)
- The sport tag is saved on the custom session for categorization

/// Derives the category from the session's sport and category fields.
SessionCategory categoryFor(SessionModel session) {
  if (!session.isPreset) return SessionCategory.custom;

  final sport = session.sport;
  final cat = session.category;

  if (sport == null && cat == 'conditioning') return SessionCategory.conditioning;

  return switch ((sport, cat)) {
    ('boxing', 'competition') => SessionCategory.boxingCompetition,
    ('boxing', 'training') => SessionCategory.boxingTraining,
    ('boxing', 'drills') => SessionCategory.boxingDrills,
    ('muayThai', 'competition') => SessionCategory.muayThaiCompetition,
    ('muayThai', 'training') => SessionCategory.muayThaiTraining,
    ('muayThai', 'drills') => SessionCategory.muayThaiDrills,
    ('mma', 'competition') => SessionCategory.mmaCompetition,
    ('mma', 'training') => SessionCategory.mmaTraining,
    ('mma', 'drills') => SessionCategory.mmaDrills,
    ('bjj', 'competition') => SessionCategory.bjjCompetition,
    ('bjj', 'training') => SessionCategory.bjjTraining,
    ('bjj', 'drills') => SessionCategory.bjjDrills,
    ('kickboxing', 'competition') => SessionCategory.kickboxingCompetition,
    ('kickboxing', 'training') => SessionCategory.kickboxingTraining,
    ('kickboxing', 'drills') => SessionCategory.kickboxingDrills,
    ('wrestling', 'competition') => SessionCategory.wrestlingCompetition,
    ('wrestling', 'training') => SessionCategory.wrestlingTraining,
    ('wrestling', 'drills') => SessionCategory.wrestlingDrills,
    _ => SessionCategory.custom,
  };
}

/// Returns the sport color for a session (for card accent bars).
Color categoryColorFor(SessionModel session) {
  final sport = session.sport;
  if (sport != null) {
    return switch (sport) {
      'boxing' => SportColors.boxing,
      'muayThai' => SportColors.muayThai,
      'mma' => SportColors.mma,
      'bjj' => SportColors.bjj,
      'kickboxing' => SportColors.kickboxing,
      'wrestling' => SportColors.wrestling,
      _ => CategoryColors.custom,
    };
  }
  if (session.category == 'conditioning') return CategoryColors.conditioning;
  return CategoryColors.custom;
}

/// Groups presets by sport, then by subcategory within each sport.
/// Returns: [ ('Boxing', [ ('Competition', [sessions]), ('Training', [sessions]), ... ]), ... ]
List<(String, List<(String, List<SessionModel>)>)> groupPresetsBySport(
    List<SessionModel> presets) {
  final sportOrder = ['boxing', 'muayThai', 'mma', 'bjj', 'kickboxing', 'wrestling', null];
  final categoryOrder = ['competition', 'training', 'drills', 'conditioning'];

  final sportGroups = <String?, Map<String?, List<SessionModel>>>{};

  for (final session in presets) {
    final sport = session.sport;
    final cat = session.category;
    sportGroups.putIfAbsent(sport, () => {});
    sportGroups[sport]!.putIfAbsent(cat, () => []);
    sportGroups[sport]![cat]!.add(session);
  }

  final result = <(String, List<(String, List<SessionModel>)>)>[];
  for (final sport in sportOrder) {
    if (!sportGroups.containsKey(sport)) continue;
    final sportLabel = switch (sport) {
      'boxing' => 'Boxing',
      'muayThai' => 'Muay Thai',
      'mma' => 'MMA',
      'bjj' => 'BJJ',
      'kickboxing' => 'Kickboxing',
      'wrestling' => 'Wrestling',
      _ => 'Conditioning',
    };
    final subcategories = <(String, List<SessionModel>)>[];
    for (final cat in categoryOrder) {
      final sessions = sportGroups[sport]![cat];
      if (sessions != null && sessions.isNotEmpty) {
        final catLabel = switch (cat) {
          'competition' => 'Competition',
          'training' => 'Training',
          'drills' => 'Drills',
          'conditioning' => 'Conditioning',
          _ => cat ?? 'Other',
        };
        subcategories.add((catLabel, sessions));
      }
    }
    if (subcategories.isNotEmpty) {
      result.add((sportLabel, subcategories));
    }
  }

  return result;
}

/// Legacy flat grouping (for backward compatibility if needed).
/// Groups by sport only (no subcategories).
List<(String, List<SessionModel>)> groupPresetsBySportFlat(
    List<SessionModel> presets) {
  return groupPresetsBySport(presets)
      .map((sportGroup) {
        final allSessions = sportGroup.$2.expand((cat) => cat.$2).toList();
        return (sportGroup.$1, allSessions);
      })
      .toList();
}

## Migration Notes

Since there are no users yet, this is a clean break. However, for robustness:

1. **Old preset IDs**: The old preset IDs (`preset_pro_boxing_men`, `preset_muay_thai`, etc.) are being replaced with new IDs (`preset_boxing_pro_men`, `preset_mt_pro`, etc.). Since presets are always loaded from code (not Hive), this is a non-issue.

2. **Custom sessions without `sport`/`category`**: Existing custom sessions stored in Hive will have `sport: null` and `category: null` by default (Freezed defaults). They will be categorized as "Custom" automatically. No migration code needed.

3. **Timer checkpoint compatibility**: The `TimerCheckpoint` stores a JSON snapshot of the session. Old checkpoints with the old model shape will still deserialize correctly because the new fields have defaults. No migration needed.
